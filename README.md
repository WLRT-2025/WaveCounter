# WaveCounter

Canonical structural framework and engineering playbooks for trading system design.

## Status
This repository contains fixed and versioned materials related to the WaveCounter system.
All canonical and consolidated documents are published in read-only form.

## Scope
WaveCounter defines structural configurations and constraints.
It does not provide:
- trading strategies
- trading signals
- performance claims
- executable code

## Usage
Materials in this repository are intended for architectural reference and system design.
Any operational, strategic, or executional decisions are external to WaveCounter.

## Repository structure
This repository serves as a canonical publication space.
Materials are added only through explicit versioned publication packs.
