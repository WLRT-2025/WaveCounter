## Version scope
This document describes the versioning rules for the consolidated
**WaveCounter Engineering Playbooks**.

## v1.0
- Status: Fixed
- Languages: EN / RU
- Contents: PB-1 through PB-5 (verbatim consolidation)
- Modification policy: Read-only

## Version progression
Any modification to wording, structure, ordering, or interpretation
requires a new version identifier (v1.1+).

Minor editorial changes without version increment are not permitted.

## Language symmetry
All fixed versions must maintain semantic and structural symmetry
across supported languages.
